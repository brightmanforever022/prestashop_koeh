Koehlert.com web service

URL: https://www.koehlert.com/web_service/index.php
All requests sent using HTTP POST method.

web_service=sale_stats_daily

Returns daily sale statistics.

Response fields:
"sold_quantity" - quantity sold
"sale_revenue" - sales revenue
"report_date" - report date

Parameters

date_year_month
Date in format YYYY-MM

web_service=sale_stats_weekly

Returns weekly sale statistics.

Response fields:
"sold_quantity" - quantity sold
"sale_revenue" - sales revenue
"report_date" - report date

Parameters

date_year
Year of the report, in YYYY

web_service=sale_stats_monthly

Returns monthly sale statistics.

Response fields:
"sold_quantity" - quantity sold
"sale_revenue" - sales revenue
"report_date" - report date

Parameters

date_year
Year of the report, in YYYY

web_service=photos

Return photos urls for all products.
